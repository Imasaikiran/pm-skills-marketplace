# Complete PM Skills Catalog

This document outlines all 60+ Product Manager skills across 12 categories for the marketplace.

## Category 1: Discovery & Research (8 skills)

### ✅ 1.1 user-interview-guide (CREATED)

- **Frameworks:** JTBD, 5 Whys, Contextual Inquiry

- **Frameworks:** Journey mapping, Touchpoint analysis, Moment mapping
- **Outputs:** Visual journey maps, pain point prioritization, opportunity identification

- **Description:** Deep dive JTBD analysis to uncover functional, emotional, and social jobs
- **Frameworks:** JTBD, Job Stories, Switch interviews

### 1.5 survey-design
- **Description:** Design effective user surveys with unbiased questions and actionable insights
- **Frameworks:** Question design best practices, Response bias mitigation
- **Outputs:** Survey templates, analysis guides, insight extraction

### 1.6 market-sizing
- **Description:** Calculate TAM, SAM, SOM using top-down and bottom-up approaches
- **Frameworks:** TAM/SAM/SOM, Market sizing methodologies
- **Outputs:** Market size estimates, growth projections, addressable market analysis

### 1.7 user-persona-builder
- **Description:** Create evidence-based user personas from research data
- **Frameworks:** Proto-personas, Empathy maps, Persona validation
- **Outputs:** Persona documents, use case scenarios, behavioral patterns

### 1.8 usability-testing-guide
- **Description:** Plan and conduct usability tests with clear success metrics

- **Outputs:** Test plans, observer guides, findings reports

## Category 3: Prioritization & Decision-Making (7 skills)

### ✅ 3.1 rice-framework (CREATED)
- **Description:** Use RICE framework (Reach, Impact, Confidence, Effort) to prioritize features
- **Frameworks:** RICE scoring, Data-driven prioritization
- **Outputs:** RICE scorecards, prioritized backlogs, justification docs

### 3.2 moscow-prioritization
- **Description:** Categorize features using MoSCoW (Must/Should/Could/Won't have)
- **Frameworks:** MoSCoW method, Timeboxed prioritization
- **Outputs:** Prioritized feature lists, sprint planning docs, stakeholder agreements

### 3.3 kano-model
- **Description:** Analyze features using Kano Model to identify delighters vs. must-haves
- **Frameworks:** Kano analysis, Customer satisfaction mapping
- **Outputs:** Kano survey templates, feature categorization, investment recommendations

### 3.4 value-vs-effort-matrix
- **Description:** Plot features on Value vs. Effort matrix to identify quick wins
- **Frameworks:** 2×2 matrix, Quick win identification, Ice box candidates
- **Outputs:** Visual matrices, prioritized quadrants, action plans

### 3.5 opportunity-scoring
- **Description:** Score opportunities using Opportunity Solution Trees and impact assessment
- **Frameworks:** Opportunity Solution Trees, Impact scoring, Discovery prioritization
- **Outputs:** Opportunity trees, scored opportunities, discovery roadmaps

### 3.6 ice-scoring
- **Description:** Use ICE (Impact, Confidence, Ease) for rapid prioritization
- **Frameworks:** ICE scoring, Rapid prioritization
- **Outputs:** ICE scorecards, ranked initiatives, experiment queues

### 3.7 cost-of-delay
- **Description:** Calculate Cost of Delay to prioritize based on economic impact
- **Frameworks:** CD3 (Cost of Delay Divided by Duration), WSJF
- **Outputs:** Cost of delay calculations, economic prioritization, urgency analysis

## Premium Tier Skills (Proposed)

These skills would be available only in the Pro/Team tiers:

1. Strategic-level skills (vision, positioning, north-star)
2. Advanced analytics (cohort analysis, LTV:CAC, ab-test-design)
3. Leadership skills (executive presence, conflict resolution, mentorship)
4. Specialized skills (API documentation, architecture collaboration)
5. Custom variations of frameworks (weighted RICE, industry-specific OKRs)

**Freemium would include:**
- Core execution skills (backlog, stories, sprint planning)
- Basic prioritization (RICE, MoSCoW, Value vs. Effort)
- Foundational research (user interviews, surveys, personas)
- Basic metrics (AARRR, funnel analysis)
- Basic roadmapping (Now-Next-Later)
